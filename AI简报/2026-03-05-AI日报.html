Meta 发布 Code Llama 3.1 系列：专为代码生成优化的 70B 和 405B 参数模型
📅 发布于 2026-03-05
核心内容： Meta AI 正式推出 Code Llama 3.1 系列，包含 70B 和 405B 参数两个版本。该系列基于 Llama 3.1 架构，在大量代码数据上进行了专门训练和优化，显著提升了代码生成、补全、调试和解释的能力。模型支持多种编程语言，并针对长上下文代码理解和推理进行了增强。
价值： 这是 Meta 在代码专用大模型领域的重要更新，405B 参数版本有望在复杂编程任务上达到新的性能基准，为开发者和企业提供强大的 AI 编程助手。

Google DeepMind 发布 AlphaFold 3.1，显著提升蛋白质-小分子复合物预测精度
📅 发布于 2026-03-05
核心内容： Google DeepMind 宣布推出 AlphaFold 3.1 版本，这是对去年发布的 AlphaFold 3 的重大升级。新版本在预测蛋白质与药物小分子、核酸等配体相互作用的精度和速度上均有显著提升，并扩展了对更复杂生物分子系统的建模能力，为药物发现和基础生命科学研究提供了更强大的工具。
价值： AlphaFold 系列的持续迭代巩固了其在结构生物学 AI 领域的领导地位，3.1 版本对药物研发具有直接的加速作用，是 AI for Science 的里程碑式进展。

Anthropic 推出 Claude 3.5 Sonnet 的“长上下文工作记忆”实验性功能
📅 发布于 2026-03-04
核心内容： Anthropic 为 Claude 3.5 Sonnet 模型推出了一项名为“工作记忆”的实验性功能。该功能允许模型在长达数小时的单次对话中，更持久、更一致地记住用户提供的文件内容、指令和对话历史，显著提升了处理超长文档和复杂多步骤任务的能力，目前面向部分 Claude Pro 用户开放测试。
价值： 这项功能直接针对当前大模型在长上下文应用中的核心痛点（记忆丢失、不一致），是迈向更实用、更可靠 AI Agent 的关键一步，具有重要的产品应用价值。
